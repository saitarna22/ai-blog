# 運用コスト試算

最終更新: 2026-02-09

## 前提

- OpenAI モデル: gpt-4o（テキスト）、DALL-E 3（画像 1792x1024 standard）
- Firebase: Blaze プラン（従量課金 + 無料枠）
- Vercel: Hobby プラン（無料）
- 1投稿あたりのAPIコール: テキスト生成(1回) + ストーリーライン更新(1回) + 画像生成(1回)

## コスト削減オプション

| 施策 | 効果 |
|------|------|
| 画像サイズを 1024x1024 に変更 | DALL-E 3 が $0.040/枚に半減（パターンA: -$1.92、パターンB: -$3.60） |
| gpt-4o-mini に変更 | テキスト費用が約1/10に（品質トレードオフあり） |
| ストーリーライン更新を gpt-4o-mini に変更 | ストーリーライン部分のみ ~1/10 に（本文品質は維持） |
| 画像生成を隔回にする | DALL-E 3 コストが半減 |

## 注意事項

- **Vercel Hobby の Function タイムアウト**: 10秒制限。OpenAI のテキスト+画像生成は合計10秒を超える可能性あり。タイムアウトが頻発する場合は Vercel Pro（$20/月）が必要
- **Firebase Storage 累積**: 画像は削除しない限り累積する。5GB超過後は $0.026/GB/月が発生
- **読者数増加時**: Firestore リード数・Storage DL帯域が増加するが、月間数万PV程度なら無料枠内
- **為替レート**: 1ドル = 約147円で換算（変動あり）
